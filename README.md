# tinyapp-template
template for alipay miniapp

生成的目录结构：
```
+----scss  // 存放scss源码，目录结构对应小程序的目
录结构
  +---- pages
    -- home.scss
    -- other page.scss
  app.scss
+----source  // 存放小程序目录
  +----pages  
    +-- home  // 首页
    +-- other page
  --- app.acss
  --- app.js
  --- app.json

--- gulpfile.js  // gulp打包配置文件
--- package.json  // 配置文件，项目依赖
```
在`支付宝开发者工具`中，只需要打开source目录即可。
